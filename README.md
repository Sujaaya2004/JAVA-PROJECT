# JAVA-PROJECT

==> Database Connection Manager (Singleton Pattern in Java)
-> Project Overview

This project demonstrates the use of the Singleton Design Pattern in Java by creating a single, globally accessible Database Connection Manager.
It mimics a real-world backend setup where only one instance of a resource (like a database connection) should exist across multiple services.

-> Tech Stack
>>Java 17+
>>JDK + VS Code / IntelliJ IDEA / Eclipse
